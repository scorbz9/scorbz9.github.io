Everswole

Everswole is a clone of Evernote geared specifically towards tracking your weekly exercises. It uses a React/Redux frontend to create a single-page app experience.

Whelp

Whelp is a clone of Yelp for all things puppy-related. It was built with responsive re-sizing, allowing mobile users to navigate the site quickly and easily, and also features Google Maps API integration and AWS photo upload.

Lairbnb

Lairbnb is a clone of Airbnb for spooky houses and lairs. With Lairbnb, I got the opportunity to experiment with the Google Maps API as well as AWS integration for photo upload.

Jolly Jingles

Jolly Jingles is a clone of GoodReads featuring exclusively happy, up-beat tunes. It employs a PostgreSQL join table to allow users to save their favorite songs to lists.
